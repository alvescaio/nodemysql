# nodemysql
Utilização de BD mysql com nodejs

# Módulos Node utilizados
* mysql
```
$ npm install -S mysql
```
* express
```
$ npm install -S express
```
* body-parser
```
$ npm install -S body-parser
```
* sequelize
```
$ npm install -S sequelize
```
